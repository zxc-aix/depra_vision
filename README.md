# depra_vision
## Video-Based Laboratory Animal Behavior Analysis Tool

This software provides automated analysis of laboratory animal behavior from video recordings. It supports several common behavioral tests, including the Open Field Test, Y-Maze Test, and Morris Water Maze Test. The tool generates trajectory maps and heatmaps to visualize animal movement and activity patterns.

## Features

- **Open Field Test**: Analyzes locomotion, exploration, and anxiety-like behavior in an open arena.
- **Y-Maze Test**: Evaluates spatial working memory and alternation behavior.
- **Morris Water Maze Test**: Assesses spatial learning and memory based on escape latency and path efficiency.
- **Trajectory Mapping**: Plots the animal's path over time.
- **Heatmap Generation**: Creates density maps showing areas of high activity.
- **Batch Processing**: Analyze multiple videos simultaneously (if applicable).
- **Customizable Parameters**: Adjust arena zones, thresholds, and output settings.
